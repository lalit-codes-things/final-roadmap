Streaming & Observability

  • Apache Flink: event-time processing, stateful operations
  • Real-time dashboards: Grafana + Loki + Tempo
  • Build a fault-tolerant streaming pipeline (Kafka -> Flink -> Delta Lake)
  • SLI/SLO design for data pipelines

Serving Systems

  • Prefill/decode separation with vLLM as the scheduler entry point
  • Build a small benchmark for cache transfer time versus compute time
  • Study Mooncake before writing the toy disaggregated serving demo
  • Track when KV cache movement, not model compute, becomes the bottleneck

Agentic Serving

  • Reframe the Claude + MCP system as an agentic serving workload
  • Measure request fan-out, retries, tool loops, and cache reuse
  • Read AgentServe or ThunderAgent before finalizing the design
  • Optimise for many small calls instead of one long generation

Data Modeling & Storage Systems

  • Relational: normalization, foreign keys, indexes, query optimization
  • NoSQL: document (MongoDB), key-value (Redis), columnar (Cassandra)
  • Distributed storage: S3 internals, HDFS, Delta Lake
  • Read: "The Tail at Scale" (Jeff Dean, Google)

Site Reliability Engineering (SRE)

  • SLIs: latency, error rate, throughput, saturation
  • SLOs: target values (e.g., 99.9% uptime)
  • Error budgets: how much downtime you can afford
  • Blameless post-mortems: learn from incidents, never blame
  • Read: Google's SRE Book (Ch. 2-4)
